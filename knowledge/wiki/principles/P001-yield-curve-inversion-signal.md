## 解读

- 倒挂持续 3 个月以上（而非单日倒挂）才是可靠信号
- 10Y-3M 利差的预测能力比 10Y-2Y 更强（纽约联储 Probit 模型）
- 倒挂解除（恢复正斜率）后约 6-24 个月内迎来衰退——"最后的绿灯"
- QE 期间人为压低长端，需调整期限溢价后判断

## 历史案例

| 倒挂开始 | 衰退开始 | 领先时间 |
|---------|---------|---------|
| 2006-08 | 2007-12 | 16 个月 |
| 2000-07 | 2001-03 | 8 个月 |
| 1989-05 | 1990-07 | 14 个月 |
| 1980-09 | 1981-07 | 10 个月 |
| 1978-08 | 1980-01 | 17 个月 |

## 回测结果（2026-04-25）

参见 `lab/tools/backtest_principles.py` `backtest_p001()`

**数据范围**：1995-01 至 2026-04（T10Y2Y 日数据）

| 指标 | 结果 |
|------|------|
| 有效信号（≥30天持续倒挂） | 4 次 |
| 正确预测衰退 | 3 次 |
| 误报 | 1 次（2022-2024 倒挂，尚未衰退，待验证） |
| 遗漏衰退 | 2 次（1990 数据前，2020 COVID 外生冲击） |
| 精确率 | 75%（3/4） |
| 召回率 | 60%（3/5，除去 COVID 为 100%） |
| 平均领先时间 | 17.3 个月 |
| 领先范围 | 13-23 个月 |

**结论**：原则基本成立。2022-2024 的持续 783 天深度倒挂（最大 -1.08%）是当前最大的未经验证的测试案例。COVID 衰退是外生冲击，不应算作"遗漏"。

## 局限

- 倒挂领先时间波动大（6-24 个月），无法精确判断衰退时点
- 2019 年倒挂后出现疫情——属于外生冲击，不是内生衰退
- 美联储 QE 期间信号可能失真
