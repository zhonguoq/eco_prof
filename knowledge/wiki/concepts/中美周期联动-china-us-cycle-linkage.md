# 中美周期联动 (China-US Cycle Linkage)

## 定义

全球最大的两个经济体之间的周期传导机制——包括贸易通道、金融通道和预期通道。理解中美周期联动是判断全球宏观走势的关键前提，也是 eco_prof 框架从单一美国市场扩展到全球配置的核心一环。

---

## 传导通道

### 通道一：贸易通道（传统最强）

```
美国需求变化 → 中国出口变化 → 中国制造业 PMI → 中国 GDP
          ↑                                      │
          └──── 中国生产 → 全球商品价格 → 美国 CPI ──┘
```

**历史相关性**：
- 美国 ISM 制造业 PMI 与中国出口增速：相关系数 ~0.7
- 中国 PPI 与美国 CPI 核心商品：相关系数 ~0.6
- 但相关性在 2018 年贸易战后**系统性下降**

**当前状态（2026）**：
- 伊朗战争重塑全球能源流，美国能源自给率提升 → 贸易通道进一步弱化
- 美国"友岸外包"（墨西哥/越南/印度取代中国份额）→ 结构性脱钩
- 中国对美出口依赖度从 2018 年 ~20% 降至 ~15%

### 通道二：金融通道（上升中）

```
美国利率变化 → 中美利差 → 资本流动 → USDCNY 汇率 → A 股外资流入
```

**机制**：
- 美联储加息 → 中美利差扩大 → 资本流出中国 → 人民币贬值压力 → A 股承压
- 反之美联储降息 → 中美利差收窄 → 资金回流中国 → A 股受益

**但此通道非对称**：
- 中国有资本管制，外资在 A 股占比仅 ~3%（北向资金）
- 人民币汇率有中间价管控，非完全市场化
- 因此金融通道的传导比发达国家之间**弱得多**

### 通道三：预期通道（最不可测）

```
美国政策变化 → 全球风险偏好 → 中国投资者情绪 → A 股波动
```

这是最弱但也最不可忽视的通道——美股大跌时日，A 股往往跟随低开，但可能日内反弹（"独立行情"）。

### 通道四：科技供应链通道（Decoupling）

```
美国技术出口管制 → 中国芯片/半导体/AI 自主化 → 相关板块主题行情
          │
          └──→ 中国被迫投入更多资源于自主可控 → 短期效率损失
```

**当前状态**：美国对华技术管制持续加码，中国在成熟制程芯片（28nm+）取得突破，但先进制程（7nm 以下）仍依赖受限设备。

## 当前中美异步周期（2026）

| 维度 | 美国 | 中国 | 含义 |
|------|------|------|------|
| 增长 | 1.99%（低） | 4.98%（中） | 中国增速优势仍在，但差距缩小 |
| 通胀 | 3.32%（高） | 0.22%（低） | 美国通胀 vs 中国通缩——根本性差异 |
| 利率方向 | Fed 降息预期（但通胀限制空间） | 央行持续宽松（已降准多次） | 利差扩大空间有限 |
| 汇率 | DXY 118（偏强） | USDCNY 6.82（稳定偏弱） | 人民币有贬值压力但被管控 |
| 债务周期 | 早期健康（5 信号全 ok） | 泡沫后去杠杆（房地产债务危机） | 美国在周期上半段，中国在下半段 |
| 资产偏好 | 现金++ / 商品+ / 股票-- | 债券+ / 高股息+ / 成长- | 截然不同的配置方向 |

**核心结论**：中美处于大债务周期的**不同相位**——这是全球配置最大的 Alpha 来源。

## 历史对标

| 时期 | 美国相位 | 中国相位 | 全球资产表现 |
|------|---------|---------|------------|
| 2008-2009 | 金融危机/通缩 | 4万亿刺激/信贷扩张 | 中国资产跑赢（2009 年 A 股 +80%）|
| 2014-2015 | 复苏/QE 退出 | A 股泡沫/去杠杆 | 中美不同步，A 股独立行情 |
| 2018-2019 | 加息末期/贸易战 | 贸易战/去杠杆 | 股债双杀（两国同步下跌） |
| 2020-2021 | 疫情放水/通胀起 | 疫情后出口繁荣 | 中国先受益后受损（政策转弯） |
| 2022-2024 | 激进加息/倒挂 | 房地产危机/通缩 | 美股 AI 牛 vs  A 股熊 |
| **2026 当前** | **Stagflation** | **通缩去杠杆** | **最异步相位** |

## 对 eco_prof 框架的含义

### 1. 资产配置多样性

中美异步周期意味着跨市场配置可以有效降低组合波动。当美国 Stagflation（股票/债券双杀）时，中国资产可能独立表现——但这需要中国自身周期处于有利相位（当前中国处于去杠杆阶段，自身也有压力）。

### 2. 原则的适用性

| 原则 | 在中国适用？ | 修正 |
|------|------------|------|
| P001（收益率曲线） | ❌ | 中国国债利率非市场化，无法作为领先指标 |
| P002（名义增速 vs 利率） | ⚠️ 部分 | 国企对利率不敏感，但方向性判断仍有参考价值 |
| P003（债务/GDP 阈值） | ⚠️ 参考 | 中国 ~300%但结构不同（国企债务风险低），需调整阈值 |
| P004（Regime 资产映射） | ❌ | 四象限模型的通胀/资产映射完全不同（见 A 股特征页）|
| P005（资产通胀背离） | ⚠️ 参考 | A 股散户主导下背离模式不同 |

### 3. 中国独立分析的必要性

**不应将中国资产简单视为"美国框架的延伸"**——其政策驱动、散户主导、资本管制的特征要求独立分析框架。

建议未来为 eco_prof 增加：
- 独立的中国 regime 判断模块（基于不同阈值和资产映射）
- 中美周期相位对比 → 全球配置建议
- 中国政策脉冲的定性分析能力

## 局限

- 去美元化趋势可能改变金融通道的传导机制
- 政治风险（台海）是最大尾部因素，无法建模
- 中国数据透明度有限，研判置信度天然较低
- 跟踪中美国债利差等被广泛关注的指标可能已过度定价
