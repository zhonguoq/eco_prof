# /eco-chat — 分析对话模式

进入深度分析对话模式。加载 eco-prof 主编排 Agent，获得所有 Skills 的完整能力。

## 在这个模式下你可以

- **deep dive**：就某个话题深入追问（如"详细分析当前收益率曲线的含义"）
- **what-if**：情景推演（如"如果伊朗和谈破裂对油价的影响"）
- **compare**：对比分析（如"当前周期和 2008 年有什么异同"）
- **learn**：学习知识库内容
- **debate**：让系统先给出一个判断，你再提出反对意见，系统会基于框架反驳你或修正自己

## 使用方式

```
/eco-chat
eco-chat> 分析当前 stagflation 对科技股的影响
eco-chat> 如果美联储意外降息呢？
```

输入 `exit` 或 `back` 退出对话模式。
