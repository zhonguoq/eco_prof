授权级别：L0

用户查询经济概念、框架、历史周期、原则卡片时触发。

## 执行流程

1. 解析用户查询，确定所属分类（concept / thinker / source / analysis / principle）
2. 读取 `knowledge/wiki/` 对应目录下的文件
3. 解析文件的 frontmatter + 正文
4. 跨目录交叉关联（如概念引用思想家、分析引用原则）
5. 返回结构化答案

## 输出格式

### 概念定义
- 名称 + 简述（1-2 句）
- 关联框架（如 Dalio 债务周期、奥派商业周期等）
- 关键指标（如适用）

### 关联原则卡片
- 列出所有直接关联的原则卡片（P00X）
- 含原则的规则摘要

### 来源引用
- 来源：wiki 文件路径
- 原始材料：knowledge/raw/ 引用（如适用）
- 最后更新时间（文件 frontmatter 中的 updated）

## 约束

- **只读操作**（L0），不修改 `knowledge/` 中的任何内容
- 引用来源时必须给出 wiki 文件路径，不可杜撰
- 若知识库中不存在用户查询的内容，明确告知"知识库中无此条目"
