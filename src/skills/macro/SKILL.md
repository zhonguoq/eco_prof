name: macro
description: 宏观引擎 — 诊断 + 建模 + 回测
---

授权级别：L1

用户需要宏观诊断、债务周期阶段判断、6个信号维度查询时触发。

## 执行流程

1. 调用 `python lab/scripts/diagnose.py` 运行宏观诊断
2. 解析返回的 JSON 输出
3. 呈现阶段结论 + 信号面板
4. 若用户需要图表，追加 `python lab/scripts/diagnose.py --chart` 生成 HTML 报告

## 输出格式

### 阶段结论
- 当前阶段编号（Phase 1-7）+ 阶段名称
- 阶段描述（引用 Dalio 定义 + 用户自定义描述）
- 置信度（高/中/低 + 置信度依据简述）

### 6 维信号面板

| 维度 | 当前值 | 历史趋势 | 信号方向 |
|------|--------|----------|----------|
| 收益率曲线形状（含轨迹） | ... | ... | 看多/看空/中性 |
| CPI YoY | ... | ... | 看多/看空/中性 |
| 失业率 | ... | ... | 看多/看空/中性 |
| 消费者信心 | ... | ... | 看多/看空/中性 |
| 制造业指数 | ... | ... | 看多/看空/中性 |
| 名义增长 vs 名义利率 | ... | ... | 看多/看空/中性 |

### 结构风险叠加
- 总债务/GDP（告警 ≥ 300%, 危险 ≥ 350%）
- 联邦债务/GDP（告警 ≥ 100%, 危险 ≥ 130%）
- USD 贸易加权指数趋势

### 原则卡片引用
- 列出与当前诊断直接相关的原则卡片（P00X）
- 列出当前应用的规则（rules.json 中的阈值）

## 约束

- AI 不生成算法代码，宏观诊断由预构建 `diagnose.py` 执行
- 所有阈值来自 `lab/engine/macro/rules.json`（或 TOML 规则文件），AI 不猜测默认值
- `--chart` 参数可选，仅在用户要求图表时使用
