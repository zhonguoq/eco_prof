# 宏观环境判断与投资指引框架

> 这个页面记录**宏观环境监测仪表盘**（`lab/dashboard/`）的设计思路。
> 仪表盘是这套理论的**实时执行层**——自动拉取数据、运行判断、记录历史。

---

## 一、为什么需要这个框架

Wiki 中已有多个周期理论（[大债务周期](../concepts/大债务周期-big-debt-cycle.md)、[世界秩序大周期](../concepts/世界秩序大周期-big-cycle-of-world-order.md)、[内部秩序周期](../concepts/内部秩序周期-internal-order-cycle.md)、[储备货币周期](../concepts/储备货币周期-reserve-currency-cycle.md)），但它们是**诊断工具**，不是**决策工具**。投资决策需要回答两个问题：

1. **现在处于什么阶段？**（诊断）
2. **这个阶段应该持有什么资产？**（行动）

这个框架把两者连起来：多层周期定位 → 增长-通胀象限识别 → 资产配置倾向。

---

## 二、三层诊断结构

### Layer 1：短期债务周期（5-10 年）

**判断工具：** [债务周期阶段判断框架](./债务周期阶段判断框架.md)

**核心信号：** 收益率曲线、利率方向、名义增速 vs 利率、HY 信用利差、信用卡违约率

**回答的问题：** 我们在泡沫、顶部、萧条还是正常化？

### Layer 2：增长-通胀象限（Regime）

**模型：** 四象限经典宏观模型

```
                高增长              低增长
低通胀     Goldilocks          Deflation/通缩衰退
高通胀     Overheating/过热     Stagflation/滞胀
```

**判断变量：**
- **增长轴**：实际 GDP 同比（阈值 2%，可调）
- **通胀轴**：CPI 同比（阈值 3%，可调）
- **辅助信号**：失业率方向、消费者信心、美元指数

**回答的问题：** 当前的宏观流动性环境是什么？

### Layer 3：长期结构性风险

**核心指标：**
- 总债务/GDP（警戒线 300%，危险线 350%）
- 联邦债务/GDP（警戒线 100%）
- 美元贸易加权指数趋势（储备货币周期信号）

**回答的问题：** 我们是否在世界秩序大周期的尾段？黑天鹅的尾部风险有多大？

---

## 三、四象限 × 资产配置映射

基于 Dalio 全天候（All Weather）思路——**不同资产在不同 regime 有不同表现**，核心逻辑：

| Regime | 股票 | 长期国债 | 商品/黄金 | 现金 | 原理 |
|--------|------|---------|----------|------|------|
| **Goldilocks** | ++ | + | - | - | 企业盈利扩张，利率温和，风险资产主导 |
| **Overheating** | + | -- | ++ | - | 通胀预期推高商品，侵蚀固收购买力 |
| **Stagflation** | -- | - | + | ++ | 增长下滑 + 通胀高，股债双杀，现金+实物保值 |
| **Deflation** | - | ++ | -- | + | 央行降息，长债受益，商品需求塌陷 |

倾向编码：`++ = +2`（强配置）、`+ = +1`、`- = -1`、`-- = -2`（强低配）

### 注意事项

- **这是方向，不是仓位**。仪表盘给出的是倾向信号，具体权重需要结合个人风险偏好、税务、流动性需求来决定。
- **阈值不是金科玉律**。2% 增长 / 3% 通胀是经验值，可以在 `lab/dashboard/backend/regime.py` 中调整。
- **Regime 会误判**。数据有滞后（CPI 月度、GDP 季度），象限切换时会有一段模糊期。看**信号的持续性**，不是单次读数。

## 四、监测系统架构

仪表盘（`lab/dashboard/`）把这个框架自动化：

```
┌─────────────────────────────────────────┐
│         APScheduler (每日 06:00 UTC)     │
└──────────────┬──────────────────────────┘
               ↓ 触发
┌─────────────────────────────────────────┐
│  fetch_us_indicators.py (FRED API)      │
│  → 18 个指标 CSV + 1 个 snapshot         │
└──────────────┬──────────────────────────┘
               ↓ 调用
┌─────────────────────────────────────────┐
│  regime.py                              │
│  ├ compute_diagnosis (Layer 1)          │
│  ├ compute_regime    (Layer 2 + 3)      │
│  └ append_history    → JSONL 日志        │
└──────────────┬──────────────────────────┘
               ↓ 暴露 API
┌─────────────────────────────────────────┐
│  FastAPI                                │
│  ├ GET /api/diagnosis                   │
│  ├ GET /api/regime                      │
│  ├ GET /api/diagnosis/history           │
│  └ POST /api/refresh (手动触发)          │
└──────────────┬──────────────────────────┘
               ↓ 消费
┌─────────────────────────────────────────┐
│  React Dashboard                        │
│  ├ DiagnosisPanel   (短期债务周期)       │
│  ├ RegimePanel      (四象限 + 资产倾向)   │
│  ├ KpiCards + Charts                    │
│  └ RegimeTimeline   (历史变迁)           │
└─────────────────────────────────────────┘
```

### 核心指标清单（18 个系列）

| 层级 | 系列 ID | 说明 |
|------|---------|------|
| Layer 1 · 债务健康 | TCMDO, GDP, GFDEGDQ188S | 总债务/GDP、联邦债务/GDP |
| Layer 1 · 货币政策 | FEDFUNDS, DGS2, DGS10, T10Y2Y | 利率走廊、收益率曲线 |
| Layer 1 · 领先预警 | BAMLH0A0HYM2, DRCCLACBS | HY 利差、信用卡违约率 |
| Layer 2 · 增长 | GDPC1, RGDP_YOY | 实际 GDP 及其同比 |
| Layer 2 · 通胀 | CPIAUCSL, CPI_YOY | CPI 及其同比 |
| Layer 2 · 辅助 | UNRATE, UMCSENT | 失业率、消费者信心 |
| Layer 3 · 储备货币 | DTWEXBGS | 美元贸易加权指数 |

### 历史日志（`lab/data/diagnosis_history.jsonl`）

每天一条记录，JSONL 格式。包含：
- 诊断日期
- 短期债务周期阶段判断
- Regime 象限
- 关键数值（GDP 增速、CPI、资产倾向）
- 长期结构性风险评估

**用途：**
- **人类**：在 Dashboard 的 RegimeTimeline 中看到 regime 变化
- **LLM**：未来对话中，LLM 可以直接读这个文件，快速了解最近几个月的 regime 演变，无需重新分析原始数据

## 五、使用工作流

### 每日/每周例行检查

1. 打开 Dashboard（`http://localhost:5173`）
2. 看 Regime 面板的当前象限
3. 如果象限没变化 → 维持配置
4. 如果象限切换 → 研究切换的持续性（假信号？还是趋势转折？）

### 关键信号触发深度分析

当以下任一信号出现时，启动 wiki 深度分析：

| 触发信号 | 启动分析 |
|---------|---------|
| 收益率曲线倒挂 | 对照[债务周期阶段判断框架](./债务周期阶段判断框架.md)—顶部 12-18 个月倒计时 |
| HY 利差突破 5% | 信用市场定价恐慌—检查 Layer 1 所有信号 |
| 象限从 Goldilocks 切到 Stagflation | 增长-通胀双恶化—最糟糕的 regime 转折 |
| 总债务/GDP 突破 350% | Layer 3 警报—检查[储备货币周期](../concepts/储备货币周期-reserve-currency-cycle.md)情景 |
| 美元指数快速下跌 + 通胀上行 | 储备货币地位受挑战的典型特征 |

### 扩展方向（未来）

- 加入中国宏观指标（社融、M2、CPI）
- 加入全球风险情绪指标（VIX、黄金、BTC）
- Telegram / 邮件告警—当象限切换或关键指标突破阈值时推送

## 六、局限性

1. **只是框架，不是预测**。给出的是"当前是什么环境"，不是"接下来会怎样"。Regime 切换本身就是滞后的。
2. **只覆盖美国**。中国、欧元区、日本各有各的周期位置，全球资产配置还需跨市场比对。
3. **四象限是粗粒度**。真实市场常有"类滞胀"、"浅通缩"等中间态，阈值需要根据历史分布不断校准。
4. **没有估值层**。即便 regime 判断对，如果资产估值已充分反映，超额收益空间有限。下一步应引入估值维度（Shiller PE、期权 skew 等）。
