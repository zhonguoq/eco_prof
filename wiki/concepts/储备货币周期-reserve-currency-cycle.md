# 储备货币周期 (Reserve Currency Cycle)

## 定义

储备货币周期描述一国货币如何经由国力崛起而获得全球储备地位，又如何因国力衰退和过度举债而最终丧失这一地位。储备货币地位是帝国综合国力的"货币投影"，也是大国兴衰中最具战略意义的指标之一。

---

## 货币体系三阶段（Dalio 分类）

历史上货币体系在三种类型之间循环演进：

| 类型 | 特征 | 历史案例 |
|------|------|---------|
| **Type 1 — 硬通货** | 黄金、白银直接作为货币使用；内在价值由金属重量决定 | 古代至近代大部分时期 |
| **Type 2 — 挂钩硬通货的纸币** | 纸币是金属的凭证，承诺可按固定比率兑换；实际上是杠杆化的硬通货 | 金本位（1870-1914）、布雷顿森林体系（1944-1971） |
| **Type 3 — 法币（Fiat Money）** | 货币不与任何实物挂钩；价值来自政府信用和网络效应 | 1971 年后美元体系，当前全球主要货币 |

**周期规律**：
- 从 Type 1 → Type 2：财政/战争压力需要更多货币，开始以纸币代替金属
- 从 Type 2 → Type 3：承诺的兑换量超过黄金储备，最终放弃挂钩（Nixon Shock, 1971）
- Type 3 崩溃：持续印钞导致通胀/超通胀，信用丧失，回到 Type 1 或新 Type 2

## 储备货币获得与丧失的机制

### 获得储备地位的条件
1. 强大的经济产出和贸易网络（贸易伙伴愿意持有该货币）
2. 深度、开放的金融市场（投资者有地方放置该货币）
3. 强大的军事保障（货币背后的政治稳定性）
4. 货币的历史信用（过去没有大规模违约或贬值）

### 丧失储备地位的过程
1. **借贷阶段**：享有储备货币地位的国家可以以低成本借贷，经常出现贸易赤字和财政赤字
2. **超借阶段**：债务/GDP 比率持续攀升，对外净负债扩大
3. **印钞偿债**：当债务不可持续时，中央银行开始货币化债务（量化宽松、MMT 逻辑）
4. **货币贬值**：印钞导致货币购买力下降，持有者开始寻求替代
5. **储备地位侵蚀**：其他央行和机构降低该货币在外储中的占比
6. **骤然崩塌**：网络效应反转——一旦够多人抛弃，货币崩塌可以非常迅速

## 关键历史案例：英镑→美元的交接

**英镑的兴衰（1820s-1950s）**
- 1820s：英国工业革命后成为全球贸易/金融中心，英镑主导
- 1914-1918：一战让英国背负巨额债务（主要借自美国）
- 1944：布雷顿森林协议，美元正式取代英镑成为核心储备货币
- 1967：英国被迫贬值英镑（贬 14.3%），英镑作为储备货币地位基本结束
- **滞后期约 30-40 年**：英国工业实力在 1900 年前后开始相对衰退，但英镑储备地位维持到 1944-1967

**美元当前处境（Dalio 视角）**
- 1945：布雷顿森林建立，美元与黄金挂钩（35美元/盎司），成为全球储备货币
- 1971：Nixon Shock，美元与黄金脱钩，转入 Type 3 法币时代
- 2008 后：大规模 QE，美元供给快速扩张
- 2020：疫情后 MMT 式财政货币化，债务/GDP 超过 100%（联邦政府）
- **Dalio 警告**：美国债务轨迹与历史上进入储备货币地位丧失阶段的帝国高度吻合

## 与大债务周期的关系

储备货币周期是[大债务周期](./大债务周期-big-debt-cycle.md)在国际货币体系层面的表达：
- 大债务周期的终点（通胀型去杠杆、货币贬值）= 储备货币地位的侵蚀起点
- 不同之处：大债务周期是单国视角，储备货币周期是全球货币体系的权力转移视角
