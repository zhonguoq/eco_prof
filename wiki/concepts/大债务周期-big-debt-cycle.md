# 大债务周期 (Big Debt Cycle)

## 定义

大债务周期是由信贷扩张与收缩驱动的长期经济周期，通常历时数十年，以重大经济萧条（real GDP下降超过3%）为标志性终点。区别于短期商业周期（数年），大债务周期是多个短期债务周期的叠加——每轮短期周期的债务/收入比都比上一轮更高，直到利率无法再下调来支撑扩张为止。

> "借钱本质上是把未来的消费提前。" — Ray Dalio

---

## 七个阶段

### 1. 早期健康期（The Early Part of the Cycle）
- 债务增速与收入增速基本同步，债务被生产性地使用
- 资产负债表健康，杠杆空间充裕
- 通胀与增长"恰到好处"（Goldilocks 期）

### 2. 泡沫期（The Bubble）
- 债务增速持续**快于**收入增速
- 资产价格上涨 → 净值↑ → 抵押品价值↑ → 信贷标准放松 → 再借贷
- 影子银行系统扩张，新型金融工具涌现
- 典型特征：杠杆融资购买资产；新入场者增多；普遍乐观情绪；货币政策未能及时收紧
- 泡沫期债务/GDP比率通常以每年20-25%的速度上升，持续约3年

**识别泡沫的7个指标：**
1. 价格相对传统估值指标偏高
2. 价格已隐含对未来快速涨价的预期
3. 广泛的看涨情绪
4. 购买行为由高杠杆融资
5. 买家已大量提前购入/锁定库存
6. 新入场者大量涌入
7. 货币政策宽松有进一步吹大泡沫的风险

### 3. 顶部（The Top）
- 触发因素：央行开始收紧利率（最常见）
- 短期利率上升，收益率曲线趋平乃至倒挂
- 最脆弱的借款人开始违约，信用利差扩大
- 典型时序：收紧最快约发生在股市顶部前5个月；短期利率顶部比股市顶部早几个月

### 4. 萧条期（The Depression）
- 利率已接近0%，常规货币政策失效
- 债务违约与资产重组像雪崩式冲击杠杆率高的贷款方（尤其是银行）
- 流动性危机：金融资产/货币比率过高，大量人同时抛售
- 两类问题：**偿债能力问题**（资本不足）和**现金流问题**（流动性挤兑）
- 通缩力量主导：债务减记 + 紧缩，但缺乏足够的货币刺激对冲

### 5. 美丽去杠杆化（Beautiful Deleveraging）
→ 见专页：[美丽去杠杆化](./美丽去杠杆化-beautiful-deleveraging.md)

### 6. "推绳子"阶段（Pushing on a String）
- 货币政策进一步宽松但效果有限（"推绳子"——向前推绳子不能使绳子直）
- 经济反应迟钝，需要更直接的财政刺激配合

### 7. 正常化（Normalization）
- 债务/收入比逐步下降至可持续水平
- 信贷活动恢复正常，经济重回增长轨道

## 运作机制

**上行自我强化：**
```
借贷↑ → 支出↑ → 收入↑ → 资产价格↑ → 净值/抵押品↑ → 再借贷↑
```

**下行自我强化：**
```
违约↑ → 资产价格↓ → 净值↓ → 信用收缩 → 支出↓ → 收入↓ → 更多违约
```

---

## 短期周期 vs. 长期周期

| 维度 | 短期债务周期 | 大债务周期 |
|------|------------|----------|
| 时间跨度 | 5-8年 | 50-75年 |
| 应对工具 | 降息即可 | 降息失效，需四杆杠并用 |
| 典型结果 | 温和衰退 | 萧条（GDP下降>3%） |
| 利率底线 | 有调降空间 | 利率已接近0% |

## 四根政策杠杆

| 杠杆 | 性质 | 典型效果 |
|------|------|--------|
| 紧缩（削减支出） | 通缩性 | 减轻债务但压缩收入，效果有限 |
| 债务违约/重组 | 通缩性 | 直接减少债务存量，但引发金融恐慌 |
| 印钞/债务货币化 | 通胀性 | 提供流动性，但可能引发通胀或货币贬值 |
| 财富转移（富→穷） | 中性/再分配 | 维持社会稳定，但规模通常有限 |

## 历史案例参考

- 美国1928-1937（通缩型，大萧条）
- 德国1918-1924（通胀型，魏玛恶性通胀）
- 美国2007-2011（通缩型，大衰退，管理较好）
- 日本1989年后（通缩型，"失去的十年"）
