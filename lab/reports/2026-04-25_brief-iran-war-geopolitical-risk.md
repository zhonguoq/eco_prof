# 专题简报：伊朗战争 — 地缘风险与能源冲击

## TL;DR

伊朗战争正在重塑全球能源流向和经济格局。原油"higher for longer"已成共识，亚洲炼厂被迫减产，供应链扰动尚未完全传导到 CPI。和谈前景不明（Witkoff/Kushner 赴伊斯兰堡但伊朗称"无会面安排"），冲突升级轨道仍在。市场对地缘风险定价极不充分——美股在 AI 主题下创新高，信用利差 2.86% 仍在低位，这是当前最大的"三层背离"之外的第四层背离。

## 1. 事件概述

**当前状态**：美以对伊朗军事行动持续，伊朗试图恢复被炸机场运营

**关键时间线**：
- 美以空袭伊朗核设施/军事设施（近期）
- 伊朗试图恢复航班运行（04-25）
- Witkoff/Kushner 赴伊斯兰堡谈和（04-24），但伊朗称"无会面安排"
- 能源市场：原油"higher for longer"已成为一致预期

**参与方立场**：
| 参与方 | 立场/影响 |
|--------|----------|
| 美国 | 军事优势 + 能源自给自足受益；但同时面临全球油价上行压力 |
| 伊朗 | 制度韧性远强于委内瑞拉，单纯施压不会导致政权更迭；但基础设施受损严重 |
| 海湾国家 | 产能恢复时间不确定，油价高位受益但也面临区域不稳定 |
| 亚洲（中日韩） | 最脆弱方——炼厂减产，柴油/航煤短缺风险 |

## 2. 框架分析

### 关联原则

- **P004（Regime 资产配置）**：Stagflation 已经被当前 regime 模型确认，战争驱动的能源供给冲击将深化滞胀——CPI 从 3.3% 继续上行至 4-5% 是真实风险路径。
- **P005（资产通胀背离）**：当前 divergence = -3.56%（SP500 YoY 5.6% vs CPI 3.3%），尚未触发泡沫阈值。但值得警惕的是：消费者信心 53.3（极低位）+ 股市创新高 = 典型的"机构 vs 散户认知分化"，这本身就符合 P005 的早期预警信号。

### 对当前 Regime 的影响

```
当前：Stagflation（增长 1.99%，通胀 3.32%）
   │
   ├─ 乐观情景：伊朗和谈成功 → 油价回落 → CPI 稳定在 3-3.5%
   │    → Regime 可能维持 Stagflation 或回到 Goldilocks
   │
   ├─ 基准情景：冲突持续但有限 → 油价缓慢上行 → CPI 升至 3.5-4%
   │    → Stagflation 深化，资产配置继续倾向现金/商品
   │
   └─ 悲观情景：冲突扩大（霍尔木兹海峡/以色列全面战争）
        → 油价飙升 → CPI 突破 5% → Fed 被迫加息
        → Stagflation → 通胀型萧条风险
```
